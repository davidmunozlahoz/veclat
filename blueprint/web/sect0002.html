1 The basics

In this chapter we introduce vector lattices, vector sublattices, and ideals, and prove some basic facts.

1.1 Vector lattices

Definition 1
✓
#

Lean declarations

  • VectorLattice

A vector lattice is a real vector space \(X\) together with a lattice order \(\le \) (i.e., a partial order for which any pair of elements has a supremum and an infimum) satisfying:

  1. if \(x\le y\) and \(z \in X\), then \(x+z\le y+z\);

  2. if \(x\le y\) and \(\lambda \in \mathbb {R}_+\), then \(\lambda x\le \lambda y\).

In this chapter, \(X\) will always denote a vector lattice.

Definition 2
✓
#

Lean declarations

  • posp
  • negp

For every \(x \in X\), we define its positive part as \(x_+=x\vee 0\), its negative part as \(x_-=(-x)\vee 0\), and its absolute value as \(|x|=x\vee (-x)\).

Proposition 3
#

Lean declarations

  • nonneg_smul_sup
  • nonneg_smul_inf

For every \(x,y \in X\) and \(a \in \mathbb {R}_+\) the following hold:

\[ a(x\vee y)=(ax)\vee (ay)\text{ and }a(x\wedge y)=(ax)\wedge (ay). \]
Proof ▶

For \(a=0\) the result is direct, so assume \(a{\gt}0\). Since \(x\le x\vee y\), we have \(ax \le a(x\vee y)\). Similarly, \(ay \le a(x\vee y)\). Therefore \((ax)\vee (ay)\le a(x\vee y)\). To prove the reverse inequality, note that \(x=a^{-1}ax\le a^{-1}[(ax)\vee (ay)]\), and since the same is true of \(y\), it follows that

\[ x\vee y\le a^{-1}[(ax)\vee (ay)]. \]

Multiplying both sides by \(a\), \(a(x\vee y)\le (ax)\vee (ay)\). The equality for the infimum follows from the identity \(x\vee y=-(-x)\wedge (-y)\) that is true in every lattice ordered group.

Now we can explore some properties of the positive and negative parts.

Proposition 4
#

Lean declarations

  • posp_sub_negp
  • posp_add_negp

For every \(x \in X\):

\[ x=x_+ - x_- \text{ and }|x|=x_+ + x_-. \]
Proof ▶

For every \(a\) and \(b\) in a lattice ordered group, \(a+b=a\vee b+a\wedge b\). Putting \(a=x\) and \(b=0\):

\[ x=x\vee 0 + x\wedge 0=x_+ - (-x)\vee 0=x_+ - x_-. \]

For the absolute value, compute:

\begin{align*} x_+ + x_- & = 2x_+ - x\\ & =(2x)\vee 0 - x\\ & =x\vee (-x) & =|x|, \end{align*}

where we are using that \(c+a\vee b=(c+a)\vee (c+b)\) holds for every \(a,b,c\) in a lattice ordered group.

Lemma 5
✓
#

Lean declarations

  • leq_posp_negp

For every \(x,y \in X\), \(x\le y\) if and only if both \(x_+\le y_+\) and \(y_-\le x_-\).

Lemma 6
✓
#

Lean declarations

  • sup_of_dis_eq_sum

Let \(x,y \in X\) be such that \(x\wedge y=0\). Then \(x+y=x\vee y\).

The positive part (and therefore the negative part) is characterized by the following property.

Proposition 7
✓
#

Lean declarations

  • uniqueness_posp

Let \(x,u,v \in X\) be such that \(x=u-v\) and \(u\wedge v=0\). Then \(u=x_+\).

Next we provide some properties of the absolute value. All of them are already in mathlib but, for some reason, the first and the second are only proved under the assumption that the space is totally ordered.

Lemma 8
✓
#

Lean declarations

  • abs_eq_zero_iff_zero
  • abs_smul'

For every \(x,y \in X\) and \(a \in \mathbb {R}\):

  1. \(|x|=0\) if and only if \(x=0\);

  2. \(|ax| = |a| |x|\);

  3. \(|x+y|\le |x| + |y|\).

We are now in position to prove the Riesz decomposition theorem.

Theorem 9 Riesz decomposition
✓
#

Lean declarations

  • Riesz_decomposition

Let \(x,y,z \in X\) be positive elements satisfying \(x\le y+z\). Then there exist \(0\le x_1\le y\) and \(0\le x_2\le z\) such that \(x=x_1+x_2\).

From this point on, we will mostly deal with Archimedean vector lattices.

Definition 10
#

A vector lattice \(X\) is said to be Archimedean if for every \(x,y \in X\) with \(y{\gt}0\) there exists a natural number \(n \in \mathbb {N}\) such that \(x\le ny\).

Lemma 11
✓
#

Lean declarations

  • arch'

Let \(X\) be an Archimedean vector lattice. If \(x,y \in X\) are such that \(nx\le y\) for all \(n \in \mathbb {N}\), then \(x\le 0\).

1.2 Vector sublattices and ideals

There are two substructures of a vector lattice that will be of interest to us: vector sublattices and ideals.

Definition 12
✓
#

Lean declarations

  • VectorSublattice

A vector sublattice of \(X\) is a vector subspace \(Y\) that is at the same time a sublattice (i.e., if \(x,y \in Y\), then \(x\vee y \in Y\) and \(x\wedge y \in Y\)).

Lemma 13
✓
#

Lean declarations

  • VectorSublattice.abs_mem
  • VectorSublattice.ofSubmoduleAbs

A vector subspace of \(X\) is a vector sublattice if and only if it is closed under taking absolute values.

Definition 14
✓
#

Lean declarations

  • VectorOrderIdeal

A vector order ideal \(I\) in \(X\) is a vector sublattice such that, if \(|x|\le |y|\) and \(y \in I\), then \(x \in I\).

In the literature, vector order ideals are called order ideals or just ideals. When no risk of confusion arises, we shall do the same. However, in Lean, we have to be more careful. Ideal refers to an algebraic ideal, and order ideal refers to an ideal in a partially ordered set. Hence we have to use the full name.

Lemma 15
✓
#

Lean declarations

  • VectorOrderIdeal.mem_iff_abs_mem

Let \(I\) be an ideal in \(X\). Then \(x \in I\) if and only if \(|x| \in I\).

1.3 Principal ideals

Of special interest is the ideal generated by a single element.

Definition 16
✓
#

Lean declarations

  • PrincipalIdeal

Let \(a \in X_+\). The principal ideal generated by \(a\) is

\[ I_a = \{ \, x \in X : |x|\le s a\text{ for some }s \in \mathbb {R}_+\, \} . \]
Lemma 17
✓
#

Lean declarations

  • PrincipalIdeal.instVectorOrderIdeal

The principal order ideal generated by an element is a vector order ideal.

From now on, let \(X\) be Archimedean and let \(a \in X_+\) be fixed.

Definition 18
✓
#

Lean declarations

  • PrincipalIdeal.S
  • PrincipalIdeal.norm

For every \(x \in I_a\), define its associated norm by

\[ \| x\| _a = \inf \{ \, s \in \mathbb {R}_+ : |x| \le s a \, \} . \]

It is an important fact that this infimum is attained.

Lemma 19
✓
#

Lean declarations

  • PrincipalIdeal.norm_attained

For every \(x \in I_a\), \(|x|\le \| x\| _a a\).

We can now check that \(\| {\cdot }\| _a\) is indeed a norm.

Lemma 20
#

Lean declarations

  • PrincipalIdeal.norm_nonneg
  • PrincipalIdeal.norm_zero_iff_zero
  • PrincipalIdeal.norm_smul
  • PrincipalIdeal.norm_add

For every \(x,y \in I_a\) and \(\lambda \in \mathbb {R}\), the following hold:

  1. \(\| x\| _a=0\) if and only if \(x=0\);

  2. \(\| \lambda x\| _a= |\lambda | \| x\| _a\);

  3. \(\| x+y\| _a\le \| x\| _a+\| y\| _a\).

In fact, it is an AM-norm.

Lemma 21
✓
#

Lean declarations

  • PrincipalIdeal.norm_le
  • PrincipalIdeal.AMnorm

For every \(x,y \in I_a\):

  1. if \(|x|\le |y|\), then \(\| x\| _a\le \| y\| _a\);

  2. if \(x,y\ge 0\), then \(\| x \vee y\| _a=\| x\| _a\vee \| y\| _a\).
